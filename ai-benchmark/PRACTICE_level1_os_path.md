# Level 1: 操作系统路径操作

## 题目描述

使用 `os` 库进行路径操作，获取当前工作目录、临时目录等。

要求：
1. 获取当前工作目录
2. 获取临时目录
3. 获取主机名
4. 验证路径操作结果

## 预期输出

```
=== OS Path Operations ===
Current Directory: /path/to/current
Temp Directory: /tmp
Hostname: machine-name
✓ All tests passed!
```

## 评分标准

- 正确获取工作目录 (35%)
- 正确获取临时目录 (30%)
- 正确获取主机名 (35%)
